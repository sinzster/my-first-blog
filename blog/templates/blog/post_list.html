Database for Weld and NDE Tracking

The system would utilize a physical barcode that would be stuck on each completed weld to associate each weld with information on that weld.

  1. Welder Information: Welded By, Welded Date
  2. Final Location: Line Number, Isometric Number, Weld Number
  3. Visual Information: VT'd by, VT Date
  4. Lot Number:
  5. NDE Information: depending on % that needs to be RT/MT/UT'd
The physical barcode could also have space for "Welder Information", "Visual Information", etc. The system would also need to have offline capabilities.

Equipment Calibrations Log

A place to keep all equipment calibrations. All active/inactive calibrations. Have a sign-in sign-out method for the equipment, etc. System would also auto-notify in case there are any out of date calibrations.
